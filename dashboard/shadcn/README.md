# KOSPI shadcn dashboard

React + Vite + Tailwind dashboard styled in the shadcn spirit. It visualizes the same `reports/` artifacts generated by `python main.py` (metrics, predictions, and Dec 2025 forecast).

## Setup
```bash
cd dashboard/shadcn
npm install   # or pnpm/yarn
```

## Run (dev)
```bash
# Serves and reads reports from ../reports by default
VITE_REPORTS_BASE=../reports npm run dev
```
Open the printed localhost URL (e.g., http://localhost:5173).

## Build
```bash
VITE_REPORTS_BASE=../reports npm run build
npm run preview
```

## Notes
- Point the app at your artifacts. Easiest: create a symlink so Vite serves them:
  ```bash
  ln -snf ../../reports public/reports   # from dashboard/shadcn
  ```
  Then use `VITE_REPORTS_BASE=/reports npm run dev`. You can also copy the files into `public/reports` if symlinks arenâ€™t allowed.
- The dev server is allowed to read `../reports` via `vite.config.ts -> server.fs.allow`. Keep running `python main.py` to refresh artifacts, then hit **Reload** in the UI.
- Adjust branding/colors via `tailwind.config.ts` and `src/index.css`.
